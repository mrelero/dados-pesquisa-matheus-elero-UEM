# Aplicação de técnicas de machine learning para análise da satisfação dos clientes de banda larga no Brasil

Este repositório tem como objetivo compartilhar os dados aplicados na pesquisa realizada durante o curso de Mestrado em Engenharia de Produção da UEM. Os dados originais podem ser baixados no link: https://dados.gov.br/dados/conjuntos-dados/pesquisas-de-satisfacao-e-qualidade-percebida

## Descrição de Arquivos

- `Dados Brutos.csv`: Dados originais da pesquisa baixados diretamente do site do governo;
- `DadosProcessados.csv`: Dados com o processo de preparação realizado na pesquisa;
- `Modelo1.csv`: Base de dados preparada para o Modelo 1 aplicado na pesquisa; 
- `Modelo2.csv`: Base de dados preparada para o Modelo 2 aplicado na pesquisa; 
- `Modelo3.csv`: Base de dados preparada para o Modelo 3 aplicado na pesquisa;
